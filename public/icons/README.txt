PWA icons for iOS and Android.
Add these PNG files (use your logo or any 192x192 and 512x512 image):
- icon-192x192.png (192×192 px)
- icon-512x512.png (512×512 px)
Optional for better quality on more devices:
- icon-72x72.png, icon-96x96.png, icon-128x128.png, icon-144x144.png, icon-152x152.png, icon-384x384.png
You can generate them from one 512x512 image using a tool like https://realfavicongenerator.net or https://www.pwabuilder.com/imageGenerator
