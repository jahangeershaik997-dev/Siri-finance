Immigration & visa consultancy website – **Wizzfly Overseas** – built with Next.js 14, Tailwind CSS, React Hook Form, Framer Motion.

## Deploy on Vercel

1. Import this repo: **https://github.com/jahangeershaik997-dev/whizzfly**
2. Framework: Next.js (auto-detected)
3. Optional env var: `NEXT_PUBLIC_SITE_URL` = your live URL (e.g. `https://whizzfly.vercel.app`)
4. Deploy

Form submissions go to Formspree (configured in code). No required env vars for basic deploy.
